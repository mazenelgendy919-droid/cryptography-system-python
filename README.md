# Cryptography System (RSA & Caesar Cipher)

A robust Python implementation of classic and modern cryptographic algorithms. This project features a secure implementation of the Caesar Cipher and the RSA algorithm, designed to demonstrate core principles of Number Theory and Data Security.

## 🚀 Key Features
* **Caesar Cipher:** * Symmetric encryption/decryption using a shift-based substitution method.
    * Normalized text processing for consistent results.
* **RSA Algorithm:** * Full asymmetric encryption implementation.
    * **Prime Validation:** Built-in checks for prime number inputs.
    * **Key Generation:** Automatic calculation of the private key (d) using the **Extended Euclidean Algorithm**.
    * **Secure Modular Exponentiation:** Handles encryption and decryption of large integer values.

## 🛠️ Built With
* **Language:** Python 3.x
* **Core Concepts:** Number Theory, Modular Arithmetic, Public-key Cryptography.

## 📖 How to Use
1. Run the script: `python Cryptos-RSA-Caesar-System.py`.
2. Choose **Option 1** for Caesar Cipher: Enter your message and the shift value.
3. Choose **Option 2** for RSA: Enter two prime numbers (p, q) and a public key (e) to encrypt/decrypt messages.
